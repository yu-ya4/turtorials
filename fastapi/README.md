## Requirements

Python 3.6+

In my environment, I use `python_version = "3.7"`.

## Installation

`Pipenv`: https://github.com/pypa/pipenv 

From Nothing.

```
$ pipenv install
$ pipenv install fastapi
$ pipenv install uvicorn
```

From Pipfile.

```
$ pipenv install
```
